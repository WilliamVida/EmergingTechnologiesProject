## Getting Started
1. Clone the repo
   ```sh
   git clone https://github.com/WilliamVida/EmergingTechnologiesProject
   ```
   
### Windows
```bash
set FLASK_APP=app.py
python -m flask run
```

### Linux
```bash
export FLASK_APP=app.py
python3 -m flask run
```

Open http://127.0.0.1:5000/
